# 资料分析

$q_a$ 变化时 $\dfrac{q_a}{1+q_a}$ 的放缩原理:

1.若 $q_a$ 变大, 则 $\dfrac{q_a}{1+q_a}$ 变大

2.若 $q_a$ 变小, 则 $\dfrac{q_a}{1+q_a}$ 变小
